== Installation ==

1. Upload the entire `ai-job-listing` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings by navigating to the 'Settings' menu in WordPress.
4. Begin creating job listings from the dedicated dashboard!

== Frequently Asked Questions ==

= How do I create a job listing? =
Once the plugin is activated, navigate to the 'Job Listings' menu in the WordPress admin dashboard. From there, you can add new job listings and manage existing ones.

= Does this plugin require any external services? =
The AI description generation feature connects to the OpenAI API. This is optional and only used when you choose to generate descriptions.

= Is there a limit to how many job listings I can create? =
No, you can create as many job listings as you need.

= Can I customize the application form? =
Yes, the application form can be customized to collect the information most relevant to your hiring process.

= How do job seekers apply for positions? =
Job seekers can apply directly through your website using the application form attached to each job listing.

= Does this work with my existing theme? =
Yes, the plugin is designed to work with most WordPress themes.

== Privacy Notice ==

This plugin does not collect or store any personal data from users or site visitors by default.

If you enable the AI-based job description generation feature, job attributes (title, location, department, etc.) are sent to an external service (OpenAI) for processing. No personally identifiable information (PII) is shared.

This plugin complies with GDPR guidelines by requiring user consent via manual API key entry in the settings panel.
